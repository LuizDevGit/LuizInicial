COMANDOS GIT

git status --> mostra se tem algum arquivo para ser modificado ou que já foi alterado
git reflog --> mostra tudo o que ocorreu de comandos no seu diretório
git commit -m --> cria uma nova versão, passando um nome
git add . --> adiciona um arquivo que sofreu uma alteração
git reset --> volta uma versão do programa
git push --> envi alteração local para o remoto
git reset --hard --> volta versão de um código fonte
git branch --> cria uma branch
git checkout --> muda de branch

Passos para Mergear programas

1-git pull da branch principal
2-gerar uma nova branch a partir da branch principal
3-trabalhar e adicionar novas funcionalidades na nova branch
4-finalizar o trabalho na branch temporária
5-git checkout na branch principal
6-git pull
7-Mergiar(unir) o código da branch temporária com a
 branch principal
8-Git push da branch pricipal
