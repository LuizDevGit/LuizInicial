COMANDOS GIT

git status --> mostra se tem algum arquivo para ser modificado ou que já foi alterado
git reflog --> mostra tudo o que ocorreu de comandos no seu diretório
git commit -m --> cria uma nova versão, passando um nome
git add . --> adiciona um arquivo que sofreu uma alteração
git reset --> volta uma versão do programa
git push --> envi alteração local para o remoto
